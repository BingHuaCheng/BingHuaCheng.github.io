Adventures in Functions

C++ Inline Functions In an inline function，the compiled code is “in line” with the the other code in the program. With inline code，the program doesn’t have to jump to another location to execute the code and jump back. Inline function thus run a little faster than regular function， but they come with a memory penalty（n.惩罚）. If a program calls an inline function at ten separate locations，then the program winds up with ten copies of the function inserted into the code.
